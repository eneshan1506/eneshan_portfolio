# Eneshan Portfolio 🚀

A modern, high-performance personal portfolio built with Next.js, Tailwind CSS, and Framer Motion. Featuring a premium **Cyan-Gold** dark mode aesthetic.

## 🌟 Features

- **Premium Design:** A cohesive Cyan-Gold theme with deep dark backgrounds and elegant gradients.
- **Smooth Animations:** Powered by Framer Motion for scroll reveals, text effects, and interactive elements.
- **Responsive:** Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive UI:** Dynamic hovering effects, glassmorphism components, and a custom animated background.
- **Performance Optimized:** No heavy videos; uses lightweight CSS animations and highly optimized SVG/PNG assets.

## 🛠 Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Deployment:** Vercel

## 🚀 Getting Started

First, install the dependencies using pnpm:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app/`: Next.js App Router structure and main pages.
- `src/app/components/`: Modular React components (Hero, Skills, Projects, Navbar).
- `src/constants/`: Static data (skills lists, project data, social links).
- `src/lib/`: Utility functions and Framer Motion animation variants (`motion.js`).
